# Artificial Neural Network (ANN) Model

## Overview
This project implements an **Artificial Neural Network (ANN)** to perform predictive analysis using deep learning techniques.  
The model learns patterns from the dataset and makes predictions based on input features.

## Workflow
1. Load and preprocess the dataset  
2. Split data into training and testing sets  
3. Build an Artificial Neural Network model  
4. Train the model using training data  
5. Evaluate model performance  
6. Generate predictions  

## Model Architecture
- Input Layer  
- Hidden Layer(s) with ReLU activation  
- Output Layer for prediction

## Technologies Used
- Python  
- TensorFlow / Keras  
- NumPy  
- Pandas  
- Scikit-learn  

## Approach Type
This project uses a **Deep Learning approach**, where the neural network automatically learns relationships between input features and the target variable.
