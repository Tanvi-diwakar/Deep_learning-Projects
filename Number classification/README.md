# Handwritten Digit Classification using Neural Networks

## Overview
This project implements a deep learning model to classify handwritten digits using **TensorFlow and Keras**. The neural network learns patterns directly from image data.

## Dataset
The model uses the **MNIST dataset**, which contains **70,000 handwritten digit images (28×28 pixels)** labeled from **0–9**.

## Workflow
1. Load MNIST dataset
2. Normalize pixel values
3. Build neural network model
4. Train the model
5. Evaluate accuracy and make predictions

## Algorithms Used
- **Artificial Neural Network (ANN)**
- **Softmax Classification**

## Technologies Used
Python, TensorFlow, Keras, NumPy, Matplotlib

## Approach Type
**Deep Learning**
