# Deep Learning Projects  

## Overview  
This repository contains a collection of machine learning and deep learning projects focused on practical implementation, model building, and evaluation.

---

## Projects  

| Project Name | Description | Techniques Used | Outcome |
|-------------|------------|----------------|---------|
| Churn Prediction | Predicts customer churn based on user behavior and data | Classification, Data Preprocessing | Identifies at-risk customers |
| Handwritten Digit Classification (PCA + KNN) | Classifies handwritten digits using reduced dimensions | PCA, KNN | Efficient and faster classification |
| Number Classification | Digit recognition using ML/DL models | Neural Networks / ML Algorithms | Improved prediction accuracy |
| Graduate Admissions Prediction | Predicts admission chances based on academic profile | Regression Models | Estimates probability of admission |
