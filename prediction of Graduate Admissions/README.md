# Graduate Admissions Prediction

## Overview
This project predicts the **probability of admission for graduate programs** based on academic and profile features using machine learning techniques.

The model analyzes factors such as GRE score, TOEFL score, CGPA, and research experience to estimate admission chances.

## Dataset
The dataset contains information about student profiles including:

- GRE Score
- TOEFL Score
- University Rating
- SOP Strength
- LOR Strength
- CGPA
- Research Experience

Target Variable:
- **Chance of Admit**

## Workflow
1. Load and explore the dataset  
2. Perform data preprocessing  
3. Split dataset into training and testing sets  
4. Train machine learning model  
5. Evaluate model performance  
6. Predict admission probability  

## Algorithms Used
- **Linear Regression**

## Technologies Used
Python, NumPy, Pandas, Scikit-learn, Matplotlib

## Approach Type
**Supervised Machine Learning (Regression)**
