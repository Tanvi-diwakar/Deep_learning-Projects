# Handwritten Digit Classification using PCA and KNN

## Overview
This project builds a handwritten digit classification model using traditional machine learning techniques.  
It uses **Principal Component Analysis (PCA)** for dimensionality reduction and **K-Nearest Neighbors (KNN)** for classification.

## Dataset
The dataset contains handwritten digit images represented as **28×28 pixel values (784 features)** with labels from **0–9**.

## Workflow
1. Load and explore the dataset  
2. Scale features using StandardScaler  
3. Apply PCA for dimensionality reduction  
4. Train a KNN classifier  
5. Evaluate model performance  

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

## Key Idea
PCA reduces the high-dimensional image data, allowing the **KNN model to classify digits efficiently**.

## Approach Type
Traditional **Machine Learning** approach using feature engineering and dimensionality reduction.
